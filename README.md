# Store Health Assistant for WooCommerce

A lightweight WooCommerce plugin that scans your store for hidden issues affecting sales and SEO.

## Features

- Store Health Score
- Priority Issues
- Products without images
- Products without descriptions
- Products without price
- Out of stock products
- Low stock products
- Products never sold

3. Open "Store Health" from WordPress admin
